# 职位招聘信息

说明: 获取全国 职位招聘信息 的数据

# 获取方式: HTTP GET

# 发送参数:

type Day Month Year All

level 1: 国 2: 省 3: 市

{
	"region": "中国"
  	"position": "后端开发",// 默认所有排名第一的职位 
  	"type": "Day",  //  默认使用全部日期的数据 ，与传 "type": "All" 一样效果
  	"level": 1  // 默认 国级数据
}

# 返回数据格式: JSON

# 接口地址: https://39.106.60.145/Hbase_3.0/CountryJob_DayTop_Servlet

# 调用例子: www.baidu.com?

# 返回示例

当region为一级地区时，返回二级地区的hotRegion

当region为二级地区时，返回三级地区的hotRegion

当region为三级地区时，返回同级地区的hotRegion

new的数据为每月的新增数

rank按total排名
