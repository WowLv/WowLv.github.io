# 全国经验-薪资平均值-学历

# 获取方式: HTTP GET

# 数据格式: JSON

# 接口地址: www.baidu.com

# 调用例子: www.baidu.com?

返回数据如下：

jc： 大专及以下
rcc： 本科及以上
