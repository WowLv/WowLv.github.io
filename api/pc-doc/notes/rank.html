# 日期排行榜(date-rank)

说明: 获取全部日期 招聘信息数量的前五名

# 获取方式: HTTP GET

# 可传日期参数: date=‘2020-12-20’（该日期往回31天数据,数据是逆序的）默认不传，若要指定日期的数据可传

# 返回数据格式: JSON

# 接口地址: https://39.106.60.145/Hbase_3.0/Exp_Scalary_Servlet

# 调用例子: www.baidu.com?

# 返回示例:

{
  "success": true,
  "msg": null,
  "code": 200,
  "data": [
     {
            "name": "4/1",
            "value": 1552
        },
        {

            "value": 443
        },
        {
            "name": "3/30",
            "value": 2602
        }
    ...
  ]
}
