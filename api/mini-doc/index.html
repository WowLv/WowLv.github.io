# 职位详细信息

说明： 获取详细职位信息数据,同时返回该数据中所属城市的六项数据水平数据（salaryList、workYearList、educationList、financeStageList、companySizeList、jobNatureList）

以下为所需的数据库字段名

  • positionName
  • positionId
  • salary
  • city
  • district
  • workYear
  • education
  • jobNature
  • companyLabelList
  • companyShortName
  • positionLables
  • famousCompany
  • industryField
  • financeStage
  • companySize
  • companyLogo
  • longitude
  • latitude
  • secondType
  • thirdType
  • positionDesc

# 发送参数:

{

	"pid": 6778599

# 搜索职位信息

说明： 根据指定信息和关键字搜索返回数据

# 获取方式： HTTP GET

# 发送参数:

{
	//根据(positionName、companyShortName、secondType、thirdType)是否包含该参数而返回数据
	"key": "java"
	//根据city过滤返回的数据
	"location": "上海"[可选]
	//每次返回10条数据
	"page" : 1
	//搜索时过滤条件（Object格式）[可选]
	"filter" : {
		//这里参数名和字段名相同
		"workYear": ["应届毕业生"],
		"salary": ["10k-15k"]
		....
	}

# 相关职位匹配

说明： 根据前端收集数据匹配5条相关职位

# 获取方式： HTTP GET

# 发送参数:

"salary": "5k-8k",
"city": ["深圳"] || ["深圳", "广州", "上海"] ,
"pos": ["前端开发-web前端" ]|| ["后端开发-java", "前端开发-web前端", "产品经理-产品经理"], //当接受参数时拆分后查询secondType和thirdType
"jobNature": "全职",
"positionLables": ["java", "大数据", "云计算"] //positionLables
"extraId": ["6778599", "7006837", ... ] //查询时排除以下positionId
