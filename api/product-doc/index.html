# 市场痛点

# 市场

  • 受疫情影响，各企业元气大伤。从无论从最近曝光的各大型公司裁员信息还是技术社区中被面试题霸榜的情况，无疑把大学生毕业即失业的焦虑推上更高的高度。

# 大学生

  • 不了解市场招聘的环境（岗位是否饱和）
  • 不了解岗位需要的技术栈
  • 互联网的高速发展而学校教材落后
  • 部分老师的念 PPT 式教学法，加之脱离市场多年，技术栈落后
  • 低学历情况下的校招开发岗的可靠性

# 招聘网站

  • 招聘平台主要解决的问题，自然是对接求职者和招聘方，完成招聘求职的行为。
  • 单个招聘信息点击再过滤筛选，效率低下且可靠性不高。无法全面的分析市场环境及市场岗位技术要求

以上两点造成，学生对市场的不了解，漫无目的的学习。企业对学生的技能不满意，于是强调基础的同时也提高学历的门槛，使包括我在内的低学历学生连简历都不会被查看。

# 项目介绍

  • 基于大数据、人工智能的招聘大数据分析展示系统，利用实时、准确的热门互联网职位数据，帮助包括且不限于高校毕业生了解当前市场的招聘环境和市场对职位的技术要求。

# 项目优势

# 数据源高度可靠

  • 数据源于互联网招聘龙头企业-拉勾网

# 全面分析互联网领域的招聘信息

  • 分析从市场招聘走势到公司相关信息的覆盖市场->公司的全方面信息

# 极高的自由度

  • 数据的内容可由用户自由选择，极大的方便用户从自身的条件下查看相对应的分析

# 界面美观、易用

  • 大部分信息以图表的形式展示，能够比文字更简洁的描述出我们所想要表达的内容
  • 更好的 UI、UV。网站专业性显得更强

# 数据准确性

  • 每日凌晨 00:05 分获取昨天的招聘数据
  • 平均每日获取 9W+数据，去重后日数据新增 2000+

# 热门互联网岗位覆盖率高

  • 从测试到前后端到产品等热门互联网岗位均有覆盖

# 二次开发、迁移开发成本低

  • 完整的产品文档
  • 完整的 API 文档
  • 完整的 Github 提交记录
  • 直面用户的前端方面. 主流的前端框架 Vue,遵循组件化开发。两项导航用了多达 50 个的.vue 单文件。语法规范引用 字节跳动某前端 ESLint 语法规范
